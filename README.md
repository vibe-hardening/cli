# vibe-hardening

> **Vibe coded. Vibe hardened.**
>
> One-command security scanner for AI-generated code.

```bash
npx vibe-hardening scan
```

## What it catches

Designed for code generated by **v0 / Lovable / Bolt / Cursor / Claude Code / Replit Agent / Windsurf / Devin**.

- Leaked API keys (OpenAI, Anthropic, Stripe, GitHub, AWS, Supabase)
- Supabase tables without RLS enabled
- `NEXT_PUBLIC_*` variables leaking real secrets
- API routes without auth checks (AST-based)
- SQL / NoSQL / command injection patterns
- CORS wildcards, SSRF, XSS
- LLM-hallucinated npm packages (slopsquatting)
- Dependency CVEs via OSV.dev
